# amigoapi
A FastAPI culinary recipes website created for learning purposes.

## Functionality
* SwaggerUI docs available at `localhost:8000/docs`.
* Implemented authentication and authorization using JWT
* 

## Tech stack
* FastAPI 0.78.0
* SQLAlchemy + Alembic
* Docker
* PostgreSQL
